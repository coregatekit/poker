---

| คำ | ความหมาย |
|---|---|
| Action | การตัดสินใจในลำดับของผู้เล่น |
| All-in | ลงชิปที่เหลือทั้งหมด |
| Blind | เดิมพันบังคับก่อนแจกไพ่ |
| Board | ไพ่กองกลางที่ทุกคนใช้ร่วมกัน |
| Button / BTN | ปุ่มกำหนดตำแหน่ง dealer เชิงสัญลักษณ์ |
| Call | ลงเงินให้เท่ายอดปัจจุบัน |
| Check | ผ่านโดยไม่ลงเพิ่มเมื่อไม่มียอดต้องตาม |
| Community cards | ไพ่กองกลาง flop, turn, river |
| Fold | ทิ้งมือและสละ pot |
| Hole cards | ไพ่ส่วนตัว 2 ใบ |
| Kicker | ไพ่ประกอบที่ใช้แก้ tie ในหมวดเดียวกัน |
| Pot | ชิปกองกลางที่กำลังชิง |
| Raise | เพิ่มยอดเดิมพันปัจจุบัน |
| Showdown | เปิดไพ่ตัดสินหลังรอบสุดท้าย |
| Street | รอบแจก/เดิมพัน: pre-flop, flop, turn, river |

## จุดที่คนมักเข้าใจผิด

> “Bet” คือเปิดเดิมพันเมื่อยังไม่มียอด ส่วน “raise” คือเพิ่มยอดที่มีอยู่แล้ว แม้ภาษาพูดมักใช้แทนกัน
