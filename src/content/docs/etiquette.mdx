title: มารยาทและกฎปลีกย่อย
description: การประกาศ action, string bet, การปกป้องไพ่ และการเล่นตามลำดับ
---

เกมสดราบรื่นเมื่อทุกคนประกาศ action ชัดเจนและรอถึงลำดับของตน

- วาง raise ในครั้งเดียวหรือประกาศยอดก่อนขยับชิป เพื่อเลี่ยง **string bet**
- อย่าเปิดหรือทิ้งไพ่ก่อนถึงลำดับ เพราะข้อมูลนั้นกระทบการตัดสินใจ
- วางตัวป้องกันบนไพ่โดยไม่บดบัง เพื่อไม่ให้ Dealer เก็บผิด
- ชิปมูลค่าสูงควรมองเห็นได้เสมอ
- หากไม่แน่ใจ ให้หยุดและถาม Dealer ก่อนลงมือ

คำพูดมักมีผลผูกพัน เช่น พูด “call” แล้วจะเปลี่ยนเป็น raise ไม่ได้ แม้ยังไม่ได้ขยับชิป ทั้งนี้ใช้กฎของห้องที่กำลังเล่นเป็นหลัก

## จุดที่คนมักเข้าใจผิด

> การโยนชิปใหญ่หนึ่งชิปลงโดยไม่ประกาศ raise มักถูกนับเป็น call ไม่ใช่ raise ควรบอกเจตนาก่อนชิปแตะโต๊ะ
